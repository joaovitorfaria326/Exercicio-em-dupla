Como usar

Passo a passo

  1. Aperte F12 e abra a aba Console — tudo aparece lá.
  2. No desafios.js, cada exercício traz contexto, tarefa, exemplo, dica — e, nos mais complexos, um 🧠 como pensar passo a passo — e o resultado esperado para vocês se autocorrigirem.
  3. Piloto digita, copiloto confere — troquem a cada exercício.
  4. Salve e aperte F5 a cada mudança.

Só o desafios.js é de vocês. Este HTML e o CSS já estão prontos — não precisa (nem deve) mexer neles.

Regras do jogo

Consulta liberada, IA não

  1. ✅ Permitido: a documentação oficial (MDN) — os links do tema estão no topo do desafios.js —, o material de leitura da semana e o seu par.
  2. 🚫 Proibido: pedir a resposta para IA (ChatGPT, Claude, Gemini, Copilot…). Na SA vocês estarão sem ela — treinem como vão jogar.
  3. 🐙 Terminou? Commit nos dois repositórios do GitHub da dupla.
